Part-of-Speech Tagging(词性标注)

常见spacy词性标注

TAG(fine-grained part of speech) POS(coarse-grained part of speech) Morphology Description
NN NOUN Number = sing None,singular
NNS NOUN Number = plur None,plural
PRP PRON PronTypr = prs Pronoun,personal
PRP$ PRON PronTypr = prs
Poss = yes		 Pronoun,possessive
VB VERB VerbForm = inf Verb,base form
VBD VERB VerbForm = fin
Tense = pase		 Verb,past tense
VBG VERB VerbForm = part
Tense = pres
Aspect = prog		 Verb,gerund.or present
JJ ADJ Degree = pos Adjective

https://spacy.io/api/annotation#pos-tagging 可以找到完整表格
